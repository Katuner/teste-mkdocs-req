# Observatório de Talentos

## Visão do Produto e Projeto

Bem-vindo à documentação oficial do projeto **Observatório de Talentos — Jovem Aprendiz**, desenvolvida no contexto da disciplina de **Requisitos de Software (2026.1)** no SEST SENAT B093, Luzânia.

---

## Sobre o Projeto

O Observatório de Talentos é um **ecossistema integrado de gestão da jornada do aprendiz** no setor de transporte. Idealizado em 2026, o projeto propõe uma plataforma tecnológica com modelo de negócio sustentável que conecta o ambiente educacional presencial com os sistemas das empresas parceiras, combatendo a evasão de jovens aprendizes por meio de gamificação, acolhimento humanizado e integração de dados.

O problema central que motiva o projeto é a **taxa de evasão de aproximadamente 25% nas primeiras semanas** de aprendizagem, causada por uma combinação de fatores: ausência de acolhimento estruturado, invisibilidade do desempenho dos jovens, processos administrativos manuais e falta de integração entre os sistemas das instituições e das empresas parceiras.
